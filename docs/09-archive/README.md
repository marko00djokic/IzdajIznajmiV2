# Arhiva

Sadržaj ovde je `historical`: može objasniti raniju odluku ili incident, ali nije
aktuelni source of truth niti operativni onboarding.

- `plans/` — završeni upitnici, jednokratni planovi i zamenjeni zbirni vodiči;
- `releases/` — datirane release/hotfix beleške;
- `notices/` — istorijska obaveštenja o starim lokacijama.

Ako istorijski dokument protivreči aktivnoj dokumentaciji ili kodu, važi aktivni
source of truth. Arhivski dokument ne „osvežavaj” kao da opisuje sadašnjost;
umesto toga linkuj aktuelni dokument.
